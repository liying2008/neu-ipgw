# neu-ipgw
一键登录东北大学校园网网关
## 主要功能：
该应用用于登录东北大学校园网网关。真正实现一键登录，简单快速，避免了通过浏览器登录的繁琐操作。
## 使用说明：
用户每次只需要点击[连接网络]按钮即可登录到校园网，点击[断开网络]按钮即可断开与外网的连接。
设置默认帐户的两种方式：
1.在“网络”页面长按用户名输入框可将当前帐户设置为默认帐户。
2.在帐户列表对话框中长按某一条目可将对应的帐户设置为默认帐户。
### 其他说明：
在“网络”页面填入的用户名和密码仅当次生效。
